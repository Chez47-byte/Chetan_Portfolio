## What This Project Actually Demonstrates

Most portfolios are just UI. This one intentionally focuses on engineering discipline:

- Component-driven architecture (clean separation of concerns)
- Route-based navigation using React Router
- Smooth UI transitions using Framer Motion (without hurting performance)
- Scalable folder structure (ready for backend integration)
- Optimized build using Vite
- Responsive design without hacks or breakpoints chaos

## Core Features

- Fully responsive (mobile → desktop)
- Interactive UI with subtle animations
- Clean navigation with active route highlighting
- Reusable layout system (Header, Footer, Container)
- Structured pages: Home, About, Tech, Blog (extendable)
- Fast load times and optimized assets
- Deployed with CI-ready workflow on Vercel
